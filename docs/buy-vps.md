## 服务器配置要求

| 配置项 | 最低要求 | 说明 |
|--------|---------|------|
| 操作系统 | Ubuntu 22.04 / Debian 12 | 推荐 Ubuntu |
| 内存 | 512 MB | 足够 |
| CPU | 1 vCPU | 足够 |
| 月流量 | 500 GB+ | 看视频较多可选 1TB |
| 地区 | 亚太 | 日本/新加坡/香港延迟最低 |

## 购买流程（以 Vultr 为例）

### 1. 注册账号

访问 [vultr.com](https://vultr.com) 注册，支持支付宝/信用卡充值。

### 2. 创建服务器

1. 点击 **Deploy New Server**
2. 选择 **Cloud Compute → Shared CPU**
3. 地区选 **Tokyo** 或 **Singapore**
4. 系统选 **Ubuntu 22.04**
5. 套餐选 **Regular Cloud Compute**（$6/月，含 IPv4）
6. 点击 **Deploy Now**

### 3. 获取连接信息

等待 1-2 分钟部署完成后，记下：

- **IP 地址** — 服务器公网 IP
- **root 密码** — 用于 SSH 登录

{% hint style="warning" %}
**关于 IP 被封**：如果部署后发现 IP 无法连接（ping 不通），说明这个 IP 已被 GFW 封锁。直接销毁重建即可，Vultr 按小时计费不会多扣钱。
{% endhint %}
